World Engine scene editor

This dedicated surface is where the interactive scene editor will live. It opens in its own tab so you can build scenes without crowding the extension pane.

Current status

The editor UI is still being assembled in this testing build, but you've reached the correct entry point. Keep an eye on release notes for when the editing tools ship.

What's available now?

  • Use the main World Engine viewer to explore the bundled scene and verify settings.
  • Import/export helpers in the codebase (src/world) already serialize worlds, so custom tools can target that schema today.
  • This page will light up with hierarchy, inspector, and asset panels once the editor lands.
